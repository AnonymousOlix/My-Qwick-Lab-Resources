# Pub/Sub: Qwik Start - Console || [GSP096] ||

## Solution

### Run below code in Cloud Shell

```
gcloud services enable pubsub.googleapis.com

gcloud pubsub topics create MyTopic

gcloud pubsub subscriptions create MySub --topic=MyTopic
```
